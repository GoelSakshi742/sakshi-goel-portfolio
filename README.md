### 🏭 Cloud Manufacturing Monitoring System ⭐ NEW
**C# · ASP.NET Core · Clean Architecture · IIoT · Digital Twin · Edge Computing**

Enterprise-grade cloud manufacturing monitoring API built with Clean Architecture principles. Simulates real-time IIoT sensor data from manufacturing equipment, processes telemetry through domain-driven design layers, and exposes RESTful endpoints for dashboards and alerting.

**Architecture highlights:**
- **Clean Architecture** — Domain / Application / Infrastructure / Presentation layers fully separated
- **IIoT simulation** — machine telemetry, sensor readings, equipment health scoring
- **Digital twin concepts** — virtual representation of physical manufacturing assets
- **Edge computing ready** — designed for low-latency local processing before cloud sync
- **ASP.NET Core Web API** — versioned REST endpoints with Swagger documentation
- **Dependency injection** — repository pattern, unit of work, CQRS-inspired service layer

**Inspired by real work at Schneider Electric and Mahindra Electric** — where monitoring industrial equipment health and reducing diagnostic time were core engineering goals.

`C#` `ASP.NET Core` `.NET 9` `Clean Architecture` `IIoT` `Digital Twin` `Edge Computing` `REST API` `DDD`
